# MAR – Exportables oficiales

Este paquete contiene los exportables oficiales del sistema **Meta–Acción–Resultado (MAR)**.

## Archivo maestro

El archivo maestro es:
- `SVG/MAR_logo_full.svg`

**Todas las demás versiones derivan de él.**  
No modificar colores ni proporciones.

## Estructura del paquete

```
MAR_EXPORTS/
├─ SVG/          # Archivos fuente maestros (SVG)
├─ PNG/          # Versiones rasterizadas (PNG)
├─ MONO/         # Versiones monocromas (negro y blanco)
├─ ICONS/        # Iconos operativos
└─ README.md     # Este archivo
```

## Especificaciones técnicas

### Colores oficiales

- **Meta/Goal (Naranja)**: `#F59E0B`
- **Acción/Action (Azul)**: `#2563EB`
- **Resultado/Outcome (Plateado)**: `#D1D5DB`
- **Espiral/Separadores (Dorado)**: `#FACC15`
- **Fondo oscuro**: `#0B1220`

### Tamaños mínimos

- Logo completo: 1200 px ancho
- Símbolo: 256 px
- Icono: 64 px

### Área de seguridad

El espacio libre mínimo alrededor del logo es la **altura de la letra "M" de META**.
