# Manual de Identidad Visual

## Sistema **MAR – Meta · Acción · Resultado**

### (Goal · Action · Outcome)

---

## 1. Introducción y propósito

Este manual define de forma **normativa** la identidad visual del sistema **MAR (Meta–Acción–Resultado)**, estableciendo reglas claras para el uso de su **símbolo, logotipo, colores, tipografía y aplicaciones**.

El objetivo del manual es garantizar:

* coherencia semántica
* consistencia visual
* reproducibilidad técnica
* compatibilidad académica y profesional

---

## 2. Concepto del sistema MAR

### 2.1 Modelo conceptual

El sistema MAR se basa en una secuencia lógica y operativa:

1. **Meta (Goal)** → Dirección
2. **Acción (Action)** → Ejecución
3. **Resultado (Outcome)** → Evidencia

Este orden es **inalterable** y se refleja tanto en la estructura conceptual como en la visual.

---

### 2.2 Traducción visual del modelo

| Concepto  | Función               | Representación             |
| --------- | --------------------- | -------------------------- |
| Meta      | Intención y dirección | Color + lado del triángulo |
| Acción    | Ejecución controlada  | Color + lado del triángulo |
| Resultado | Evidencia y cierre    | Color + lado del triángulo |
| Progreso  | Iteración y mejora    | Espiral áurea              |

---

## 3. Símbolo del sistema

### 3.1 Elementos del símbolo

El símbolo MAR está compuesto por dos elementos inseparables:

1. **Triángulo**
2. **Espiral áurea (Fibonacci)**

---

### 3.2 El triángulo

El triángulo representa:

* estabilidad
* estructura
* interdependencia de los tres componentes

#### Asignación semántica y cromática

| Lado del triángulo | Concepto            | Color    |
| ------------------ | ------------------- | -------- |
| Lado 1             | Meta / Goal         | Naranja  |
| Lado 2             | Acción / Action     | Azul     |
| Lado 3             | Resultado / Outcome | Plateado |

⚠️ Esta asignación **no debe modificarse**.

---

### 3.3 La espiral áurea

La espiral representa:

* progreso acumulativo
* mejora continua
* iteración basada en resultados

**Uso exclusivo del color dorado.**  
No debe asociarse a ningún concepto MAR concreto.

---

## 4. Logotipo

### 4.1 Composición oficial

El logotipo completo se compone de:

1. Símbolo (triángulo + espiral)
2. Texto en español
3. Línea separadora
4. Texto en inglés

---

### 4.2 Texto en español (principal)

Formato obligatorio:

```
META  ACCIÓN
RESULTADO
```

* **META** → naranja
* **ACCIÓN** → azul
* **RESULTADO** → plateado

**Reglas:**

* Meta y Acción siempre en la misma línea
* Resultado siempre debajo
* El español tiene prioridad jerárquica

---

### 4.3 Línea separadora

* Línea horizontal
* Color dorado
* Mismo color que la espiral
* Separación clara entre idiomas

---

### 4.4 Texto en inglés (secundario)

Formato:

```
GOAL • ACTION • OUTCOME
```

Correspondencia cromática obligatoria:

| Inglés  | Color    |
| ------- | -------- |
| GOAL    | Naranja  |
| ACTION  | Azul     |
| OUTCOME | Plateado |

El inglés **nunca sustituye** al español.

---

## 5. Paleta cromática oficial

### 5.1 Colores principales

#### Meta / Goal — Naranja

* HEX: `#F59E0B`
* RGB: `245, 158, 11`

#### Acción / Action — Azul

* HEX: `#2563EB`
* RGB: `37, 99, 235`

#### Resultado / Outcome — Plateado

* HEX: `#D1D5DB`
* RGB: `209, 213, 219`

---

### 5.2 Colores secundarios

#### Espiral y separadores — Dorado

* HEX: `#FACC15`

#### Fondo oscuro recomendado

* HEX: `#0B1220`

#### Fondo claro alternativo

* HEX: `#F9FAFB`

---

## 6. Tipografía

### 6.1 Criterios tipográficos

* Sans serif
* Alta legibilidad
* Neutralidad académica
* Buena renderización digital

### 6.2 Tipografías recomendadas

* Inter
* Source Sans 3
* Montserrat

### 6.3 Reglas tipográficas

* Logotipo en **mayúsculas**
* Sin cursivas
* Sin subrayados
* Tracking normal o ligeramente amplio

---

## 7. Versiones del logo

### 7.1 Versiones permitidas

* Logo completo (símbolo + texto)
* Logo reducido (solo símbolo)
* Versión monocroma
* Versión flat (sin brillos)

### 7.2 Versiones no permitidas

❌ Cambiar colores  
❌ Cambiar el orden de los términos  
❌ Usar solo inglés  
❌ Añadir iconografía externa  
❌ Deformar proporciones

---

## 8. Tamaños y márgenes

### 8.1 Tamaño mínimo recomendado

* Logo completo: 1200 px ancho (digital)
* Símbolo solo: 256 px

### 8.2 Área de seguridad

* Margen mínimo: altura de la letra "M" de META
* Ningún elemento debe invadir esta área

---

* Proyectos metodológicos
